## Công nghệ sử dụng
* NodeJs : [NodeJs@18.12.1](https://nodejs.org/en/download)
* Frontend : [ReactJs@18.2.0](https://reactjs.org/)
* Backend : [ExpressJs@4.18.2](https://expressjs.com/)
* Database : [SQL](https://www.apachefriends.org/download.html)
